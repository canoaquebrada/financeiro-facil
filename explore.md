Prompt simples — Sistema de Finanças SaaS

Crie um sistema web de controle financeiro simples, moderno e responsivo, no estilo SaaS.

O sistema deve permitir que o usuário cadastre suas entradas, saídas, contas a pagar, contas a receber e acompanhe o saldo geral.

Nome do sistema

Financeiro Fácil
